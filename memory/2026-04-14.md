## 八字逻辑修复（同日上午）

### 问题
`bazi-calculator.ts` 使用 `Lunar.fromYmd()` 未传入时辰 → 时柱计算错误

### 修复
改用 `Lunar.fromYmdHms(year, month, day, hour, 0, 0)` 正确传入时辰
- 农历输入 (`isLunar=true`)：直接用 `fromYmdHms`
- 公历输入：先 `solar.getLunar()` 获取年月日，再用 `fromYmdHms` 传入时辰

### 验证

```ts
// 输出：
// ganZhi.hour: { gan: '癸', zhi: '未' } ✅
// fiveElements: { wood:0, fire:0, earth:0, metal:3, water:1 }
// shishen: ['比肩','劫财','伤官']
```

移除无用常量 `HOUR_ZHI_MAP`。
