## 修复 lunar-javascript 导入错误

### 问题
真机运行时日志显示：
```
E ReactNativeJS: '农历转换错误', [TypeError: undefined is not a function]
```

### 原因
使用 ES6 `import` 语法在 React Native 环境中可能无法正确加载 lunar-javascript 库的导出。

### 解决方案
改用 CommonJS `require()` 方式导入：

```typescript
// 修改前
import { Solar, Lunar } from 'lunar-javascript';

// 修改后
const lunar = require('lunar-javascript');
const { Solar, Lunar } = lunar;
```
